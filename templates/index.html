🏁 Node Mining Race — All 4 Nodes Compete Simultaneously

All 4 nodes (node_0–node_3) mine the same pending block at the same time using real threads. The first to find a valid nonce broadcasts its block. The others stop and adopt the winner's block via add_block(). This directly demonstrates: nonce, SHA-256, PoW condition, node competition, and broadcast.
